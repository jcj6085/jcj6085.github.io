Linux 开发环境安装

JDK

  • 官网地址

  • rpm -ivh jdk-8u172-linux-x64.rpm

  • cd /usr/java/jdk1.8.0_172-amd64

  • vim /etc/profile

  • 配置环境变量

    • export JAVA_HOME=/usr/java/jdk1.8.0_172-amd64
    • export CLASSPATH=.:$JAVA_HOME/jre/lib/rt.jar:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
    • export PATH=$JAVA_HOME/bin:$PATH

  • 使配置生效

    • source /etc/profile

Tomcat

  • tar -zxvf apache-tomcat-7.0.88.tar.gz
  • 修改conf/server.xml 增加 URIEncoding=”UTF-8”
  • 修改防火墙配置，开放8080端口
    • CentOS 6
      • vim /etc/sysconfig/iptables
      • -A INPUT -m state --state NEW -m tcp -p tcp --dport 8080 -j ACCEPT
    • CentOS 7
      • firewall-cmd --permanent --add-port=8080/tcp
  • 重启防火墙
    • CentOS 6
      • service iptables restart
    • CentOS 7
      • firewall-cmd --reload

Maven

  • tar -zxvf apache-maven-3.3.9-bin.tar.gz

  • vim /etc/profile

    • export MAVEN_HOME=/developer/apache-maven-3.3.9
    • export PATH=$MAVEN_HOME/bin:$PATH

  • source /etc/profile

  • 配置阿里云镜像

Git

  • 下载地址
  • 安装编译源码所需依赖
    • yum -y install zlib-devel openssl-devel cpio expat-devel gettext-devel curl-devel perl-ExtUtils-CBuilder perl-ExtUtils- MakeMaker
  • tar -zxvf git-2.23.0.tar.gz
  • cd git-2.23.0/
  • make prefix=/usr/local/git all
  • make prefix=/usr/local/git install
  • vim /etc/profile
    • export PATH=/usr/local/git/bin:$PATH
  • source /etc/profile
  • Git配置
    • git config --global user.name "XXX"
    • git config --global user.email "XXX@XXX"
    • git config --global core.autocrlf false
    • git config --global core.quotepath off
    • 配置代理：
      • git config --global http.proxy 'socks5://127.0.0.1:1080'
      • git config --global https.proxy 'socks5://127.0.0.1:1080'
    • 查看git所有配置
      • git config --list
    • 删除全局配置项
      • git config --global --unset user.name
    • 编辑全局配置
      • git config --global --edit
  • 配置SSH Key
    • 进入~/.ssh 查看
    • ssh-keygen -m PEM -t rsa -C "XXX@XXX"
    • cat id_rsa.pub

Docker(CentOS 7)

  • 官方文档
  • sudo yum install -y yum-utils device-mapper-persistent-data lvm2
  • sudo yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
  • sudo yum -y install docker-ce
  • sudo mkdir -p /etc/docker
  • sudo vim /etc/docker/daemon.json
    • { "registry-mirrors": ["https://4t0bob8f.mirror.aliyuncs.com\"]}
  • sudo systemctl daemon-reload
  • sudo systemctl restart docker
  • 将用户加入docker组
    • sudo gpasswd -a ${USER} docker

CentOS 7

防火墙相关

  • systemctl status firewalld #查看防火墙状态
  • firewall-cmd --list-all #查看防火墙规则
  • firewall-cmd --query-port=8080/tcp #查询端口是否开放
  • firewall-cmd --permanent --add-port=80/tcp #开放80端口
  • firewall-cmd --permanent --remove-port=80/tcp #移除80端口
  • firewall-cmd --reload #重启防火墙(修改配置后要重启防火墙)
  • iptables -S
  • https://linuxconfig.org/redhat-8-open-http-port-80-and-https-port-443-with-firewalld
  • firewall-cmd --zone=public --permanent --add-service=http
  • firewall-cmd --zone=public --permanent --add-service=https
